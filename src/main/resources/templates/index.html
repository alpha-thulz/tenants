Tenant Management System

This is the Tenant Management System application template which I created as my training ground for API testing, coming soon will be a Web and Android application.

A few bugs and defects are left out deliberately, however some not on purposes, the aim is to uncover as many bugs as possible, no fixes will be made.

Below are current registered users with access to the API, when using the API please do not use your real email address as the system does not valid if the email address is registered to anyone, the system is only for training purposes only and no real data should be used, especially personal information.

The data is not retained, on every deploy data will be lost.

Should you want to try out the API, you can check out the documentation.
